We built a bacterial flagellar motor detection system using YOLOv8 Large on Google Colab, trained on the BYU Kaggle cryo-ET dataset. Using transfer learning from pretrained COCO weights, we trained for 200 epochs with early stopping, which triggered at epoch 27, saving the best model from epoch 17. Augmentation techniques like mosaic, flipping, and rotation helped improve generalization. Our final validation results achieved a strong mAP50 of 0.875 and mAP50-95 of 0.520, with prediction confidence scores consistently above 0.70. The project successfully demonstrates that YOLOv8 can effectively localize nanoscale biological structures in complex tomographic imaging data.
